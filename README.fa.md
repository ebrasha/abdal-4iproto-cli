## 💡 چرا این نرم‌افزار ساخته شد؟

نصب و راه‌اندازی مجموعه **Abdal 4iProto** (سرور + پنل وب + ابزار تولید کلید SSH) به‌صورت دستی کاری تکراری، حساس و پر از خطاست: باید سه مخزن گیت‌هاب را بررسی کنی، فایل مناسب سیستم‌عامل و معماری را پیدا کنی، چک‌سام را بررسی کنی، کلیدهای SSH بسازی، چندین فایل JSON تنظیمات بنویسی، پورت‌های آزاد پیدا کنی و در نهایت سرویس سیستمی نصب کنی.

نرم‌افزار **Abdal 4iProto Cli** تمام این مسیر را در قالب **یک ابزار خط فرمان حرفه‌ای با تم نئون** فشرده می‌کند که هم به‌صورت **تعاملی** و هم با **آرگومان** کار می‌کند – روی **ویندوز و لینوکس**. 🎯

---

## ✨ امکانات نرم‌افزار

- 🎨 **رابط حرفه‌ای** – لوگوی ASCII سبز نئون، کادرهای رنگی، نوار پیشرفت زنده.
- 🧙 **دو حالت اجرا** – بدون آرگومان وارد منوی تعاملی می‌شوی، یا با آرگومان به‌صورت کاملاً اتوماتیک.
- 🛡️ **بررسی پورت قبل از دانلود** – پورت‌های رزروشده یا اشغال‌شده **قبل از هر دانلودی** شناسایی می‌شوند.
- 📦 **تشخیص خودکار OS/Arch** – با استفاده از `runtime.GOOS` و `runtime.GOARCH` فایل درست را انتخاب می‌کند.
- 🔐 **تأیید SHA-256** – هر فایل دانلودی با مقدار `digest` گیت‌هاب مطابقت داده می‌شود.
- 🔑 **تولید کلید SSH** – به‌صورت پیش‌فرض با `ed25519`، طول `4096` و فعال‌بودن `-force`.
- ⚙️ **نصب سرویس** – `systemd` در لینوکس و `sc` در ویندوز با اجرای خودکار در بوت.
- 👥 **مدیریت کاربران** – مشاهدهٔ لیست و جزئیات، افزودن و حذف کاربر با تعیین نقش، پهنای باند، تعداد نشست و حجم.
- 🛠️ **ویرایش زنده تنظیمات** – تغییر پورت سرور، اطلاعات ورود پنل و دیگر مقادیر در همان لحظه.
- 🔄 **ری‌استارت خودکار سرویس** – بعد از هر تغییر در تنظیمات یا کاربران، سرویس به‌صورت خودکار راه‌اندازی مجدد می‌شود.
- 🧰 **نصب خود به‌عنوان دستور سیستم** – ثبت خودش به‌عنوان دستور سراسری `abdal-4iproto-cli`.
- ⏱️ **شمارش معکوس امن** – در زمان اجرای آرگومانی، با زدن کلید `q` می‌توانی قبل از شروع، پروسه را لغو کنی.
- 🆘 **راهنمای کامل داخلی** – همهٔ سوئیچ‌ها، دستورات و عیب‌یابی فقط یک `help` فاصله دارد.

## 📖 نحوهٔ استفاده

### 🪄 حالت تعاملی

کافی است نرم‌افزار را بدون هیچ آرگومانی اجرا کنی:

```bash
abdal-4iproto-cli
```

منوی رنگی زیر نمایش داده می‌شود:

- نصب Abdal 4iProto
- حذف Abdal 4iProto
- مدیریت سرویس‌ها
- افزودن / حذف کاربر
- ویرایش تنظیمات سرور / پنل
- نصب CLI به‌عنوان دستور سیستمی
- عیب‌یابی (Diagnostics)
- راهنما (Help)

### ⚡ حالت آرگومان (مناسب اسکریپت و اتوماسیون)

```bash
# نصب کامل با پورت‌های دلخواه برای سرور و پنل
abdal-4iproto-cli install \
  --server-ports 64235,64236,64237,64238 \
  --panel-port 52202 \
  --panel-username ebrasha \
  --panel-password "your-strong-password"

# نصب فقط سرور
abdal-4iproto-cli install --server-only --key-type ed25519 --key-bits 4096 --key-force

# نصب فقط پنل
abdal-4iproto-cli install --panel-only --panel-port 52202

# افزودن کاربر
abdal-4iproto-cli user add \
  --username ali --password "secret" --role user \
  --max-sessions 2 --max-speed-kbps 512 --max-total-mb 1024

# حذف کاربر
abdal-4iproto-cli user remove --username ali

# مدیریت سرویس
abdal-4iproto-cli service status     --component server
abdal-4iproto-cli service restart    --component panel
abdal-4iproto-cli service diagnostics

# ویرایش تنظیمات
abdal-4iproto-cli config server --ports 64235,64236,64237,64238
abdal-4iproto-cli config panel  --port 52202 --username ebrasha --password "new-pass"

# نصب خود نرم‌افزار به‌عنوان دستور سراسری
abdal-4iproto-cli self-install

# حذف کامل
abdal-4iproto-cli uninstall

# راهنمای کامل
abdal-4iproto-cli help
```

### 📂 مسیرهای نصب

| سیستم‌عامل | مسیر                                                  |
|------------|-------------------------------------------------------|
| 🐧 لینوکس   | `/usr/local/abdal-4iproto-server`                     |
| 🪟 ویندوز   | `%LOCALAPPDATA%\abdal-4iproto-server`                 |

### 🔧 نام سرویس‌ها

| سیستم‌عامل | سرور                    | پنل                    |
|------------|-------------------------|------------------------|
| 🐧 لینوکس   | `abdal-4iproto-server`  | `abdal-4iproto-panel`  |
| 🪟 ویندوز   | `Abdal4iProtoServer`    | `Abdal4iProtoPanel`    |

> ⚠️ نصب سرویس و نوشتن در مسیرهای سیستمی نیاز به دسترسی **Administrator** (در ویندوز) یا **root** (در لینوکس) دارد.
